# Plataforma Digital de Gestión y Control de Personal
Este proyecto presenta el diseño y desarrollo de una Plataforma Digital de Gestión y Control de Personal, creada para optimizar la administración del personal operativo de una estación de bomberos. El sistema permite registrar usuarios, crear eventos, asignar personal, registrar horas trabajadas y calcular la nómina de manera automatizada.
El proyecto surge de la necesidad de agilizar los procesos administrativos, disminuir errores de registro manual y proporcionar un sistema confiable que garantice integridad de datos y transparencia.
